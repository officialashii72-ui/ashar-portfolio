# Images Folder

Place your portfolio images here:

- **Profile photo** - Your professional headshot
- **Project screenshots** - Images of your completed projects
- **Thumbnails** - Optimized versions for faster loading
- **Icons** - Custom icons or logos

## Recommended Image Formats

- **Photos**: JPG (optimized, 80-90% quality)
- **Graphics/Logos**: PNG or SVG
- **Thumbnails**: WebP (for modern browsers)

## Image Optimization Tips

1. Compress images before uploading (use tools like TinyPNG, ImageOptim)
2. Use appropriate dimensions (max 1920px width for full-width images)
3. Consider lazy loading for better performance
4. Use descriptive filenames (e.g., `project-ecommerce-hero.jpg`)
